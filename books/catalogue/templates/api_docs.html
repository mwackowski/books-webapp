Api documentation

  • API base endpoint listing all records: /api/books/
  • Default number of records per page is 5 and it can be increased up to 20 using page_size parameter

    • Example: /api/books/?page_size=20

  • There's also possibility to apply equality-based filtering using following keywords: title, author, language, published_date :

    Example: /api/books/?title=Hobbit&page_size=2

  • API support also field lookups: django docs

    • Example: /api/books/?title__icontains=Hobbit&published_date__lt=2018
